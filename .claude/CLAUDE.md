# プロジェクトについて

このプロジェクトは、私がGodot Engineでのゲーム開発において用いる各種機能をプラグインとして集約するプロジェクトです。

# 作業前の準備

作業を始める前に、必ず以下のファイルを読んでください。

- `PLAN.md` — そのプロジェクトで実現したいことや要件
- `SPEC.md` — 実際に採用されている仕様の詳細や、そこに至った経緯の情報
- `TASK.md` — 具体的なTODOのリスト
- `KNOWLEDGE.md` — その他、過去の失敗を繰り返さないために覚えておくべき事項
- `UI_DESIGN_RULES.md` — UIデザインを行う上でのルール
